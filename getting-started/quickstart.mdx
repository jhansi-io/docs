## Requirements

- Docker running locally
- `curl` or any HTTP client

## 1. Start Petri

```bash
git clone https://github.com/jhansi-io/petri.git
cd petri
poetry install
make run
```

Petri is now running at `http://localhost:8000`.

## 2. Create a sandbox

```bash
curl -X POST http://localhost:8000/v1/sandboxes \
  -H "Content-Type: application/json" \
  -d '{"language": "python"}'
```

```json
{
  "id": "sb_abc123",
  "language": "python",
  "status": "created"
}
```

## 3. Upload your project

Zip your project from inside the folder:

```bash
cd my_project && zip -r ../my_project.zip .
```

Upload it:

```bash
curl -X POST http://localhost:8000/v1/sandboxes/sb_abc123/upload \
  -F "file=@my_project.zip"
```

## 4. Execute it

```bash
curl -X POST http://localhost:8000/v1/sandboxes/sb_abc123/exec \
  -H "Content-Type: application/json" \
  -d '{"command": "python app.py"}'
```

```json
{
  "output": "hello from jhansi.io\n"
}
```

## 5. Run with test mode

```bash
curl -X POST http://localhost:8000/v1/sandboxes/sb_abc123/exec \
  -H "Content-Type: application/json" \
  -d '{"command": "python app.py", "test": true}'
```

```json
{
  "output": "tests/test_app.py::test_health PASSED\n1 passed in 0.11s\n"
}
```
