jhansi.io starts as a controlled execution runtime and can grow into a broader governance layer for agent actions.

## Near-term direction

- A dependable sandbox lifecycle.
- Clear SDK and MCP integration.
- Tested self-hosting paths.
- Explicit runtime and security configuration.
- Useful execution logs and diagnostics.

## Longer-term direction

- Credential mappings that reduce secret exposure to agents.
- Auditable execution and access decisions.
- Policy controls for callers, runtimes and destinations.
- Organisational and enterprise controls.

## What this page does not contain

No quarter-by-quarter commitments, unreleased product names or claims that planned controls already exist.
